Description

The module exb_plugin_log is a simple logging plugin.

Description

It logs all incoming and outgoing packets.

It should be placed before any other plugin in the chain

In it's default implementation the log plugin places all log data in it's own directory, so make sure it's writable

Commands

None

Example chat session:

  User: hello
  Bot: hello
Both messages are logged to exb/priv/plugins/log/logs/User/Year.Month.Day.Hour.log
