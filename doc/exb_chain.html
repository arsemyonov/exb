plugin_chain/1

plugin_chain(Plugins) -> any()

Creates a chain of plugins

Usage: 

 > exb_chain:plugin_chain([{default, [plugin1, plugin2, plugin3]}]).
 [{default,#Fun<exb_chain.0.64534256>}]
The resulting function is a continuation-passing style function. When invoked it calls the next plugin in chain and passes to it a new function which will call the next plugin in chain etc.
